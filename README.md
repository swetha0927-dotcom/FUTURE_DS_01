# Superstore Sales Dashboard

## Project Overview
This project analyzes Superstore sales data using Power BI and provides insights into sales performance across categories and regions.

## Tools Used
- Power BI
- CSV Dataset

## Dashboard Features
- Total Sales KPI
- Sales by Category
- Sales by Region
- Interactive Region Slicer

## Key Insights
- Technology category generated the highest sales.
- Regional sales performance can be analyzed using the slicer.
- Interactive dashboard enables dynamic filtering and analysis.
